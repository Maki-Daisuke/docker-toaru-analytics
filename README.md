とあるデータ分析用の環境を構築するDockerfileです。

JSONベースのグラフデータを分析するために使いたいツールを色々とUbuntu上にインストールします。


## Usage

    git clone https://github.com/Maki-Daisuke/toaru-analytics.git
    cd docker-toaru-analytics/
    docker build .
